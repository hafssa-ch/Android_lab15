
# 📱 LAB 15 – SQLite & Android : Gestion des Étudiants

## 🎯 Objectif

Développer une application Android simple permettant de gérer des étudiants avec une base de données SQLite locale.

---

## 🛠️ Technologies utilisées

* Java
* Android Studio
* SQLite (SQLiteOpenHelper)
* RecyclerView

---

## 📂 Architecture du projet

```
com.example.lab_15
│
├── classes
│   └── Etudiant.java
│
├── service
│   └── EtudiantService.java
│
├── util
│   └── MySQLiteHelper.java
│
├── Adapter
│   └── EtudiantAdapter.java
│
└── MainActivity.java
```

---

## ⚙️ Fonctionnalités

### ✅ Ajouter un étudiant

* Saisie du nom et prénom
* Enregistrement dans SQLite

### 🔍 Rechercher un étudiant

* Recherche par ID
* Affichage du résultat

### ❌ Supprimer un étudiant

* Suppression via ID

### 📋 Afficher la liste

* Liste dynamique avec RecyclerView

### 🔎 Filtrage

* Recherche en temps réel par nom/prénom
